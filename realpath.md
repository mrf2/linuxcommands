# `realpath` Command

**`realpath`** Resolve symbolic links and give us absolute paths

 * **`realpath`** converts any pathe (relative or symbolic) into an absolute, canonical path
 * **`realpath`** removes **`.`** and **`..`** components
 * Resolves symlinks automatically
 * Usedful for shell script, build tools, and configuration paths

## Basic Use
```bash
$realpath <filename>
```
